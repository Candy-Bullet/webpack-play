## 打包实践 (PWA)

### 主要内容

讲解了如何实现`PWA`功能，如何使用`manifest.json`创建桌面图标。

示例演示：[webpack4.x-multi-page](https://lvzhenbang.github.io/webpack4.x-multi-page/dist/)

具体说明请参考：

[pwa](https://github.com/lvzhenbang/webpack-learning/tree/master/doc/two/workbox.md)


### 运行项目

```
yarn // 安装依赖

npm run test // 开发模式

npm run build // 生产模式
```
