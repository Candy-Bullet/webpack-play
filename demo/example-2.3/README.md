## 打包实践（loader）

### 主要内容

讲解了如何使用 `extract-text-webpack-plugin` 插件从构建输出的js文件中分离出css文件。

具体说明请参考：

[extract-text-webpack-plugin](https://github.com/lvzhenbang/webpack-learning/tree/master/doc/first/extract-text-webpack-plugin.md)

```
npm run test // 开发模式

npm run build // 生产模式
```
