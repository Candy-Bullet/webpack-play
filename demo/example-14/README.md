## 打包实践 (compress-lodash)

### 主要内容

对常用的[lodash](https://github.com/lodash/lodash)进行代码压缩。

具体说明请参考：

[babel-plugin-lodash & lodash-webpack-plugin](https://github.com/lvzhenbang/webpack-learning/tree/master/doc/lodashwepbackplugin.md)

### 运行项目

```
yarn  // 安装依赖

npm run dev // 开发模式

npm run build // 发布模式
```
