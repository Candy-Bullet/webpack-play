## 打包实践（env flag）

### 主要内容

讲解了如何在webapck3.x定义一个全局变量标记来区分开发和生产模式。

具体说明请参考：

[webpack3.x env](https://github.com/lvzhenbang/webpack-learning/tree/master/doc/first/enviroment.md)

### 运行项目

```
yarn  // 安装依赖

npm run test // 开发模式

npm run build // 生产模式
```
